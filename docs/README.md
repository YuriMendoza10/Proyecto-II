# Acceso a la Documentación de OptiAcademic

La documentación oficial del Proyecto de Fin de Asignatura está centralizada
en [`Doc/`](../Doc/), organizada por fases de gestión del proyecto y con
evidencias técnicas versionables en Markdown.

Esta carpeta `docs/` funciona como punto de acceso compatible con la consigna
del repositorio PFA. No duplica el contenido oficial.
