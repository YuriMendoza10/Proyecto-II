# 1. Visión y alcance del sistema

## 1.1 Visión del sistema

**OptiAcademic** es un sistema inteligente de gestión académica orientado a la generación y optimización de horarios universitarios.

El sistema busca apoyar a administradores, coordinadores y estudiantes mediante un flujo automatizado que permita crear horarios institucionales válidos y horarios personales adaptados a las preferencias del estudiante.

## 1.2 Problema identificado

La elaboración manual de horarios académicos puede generar:

- Cruces entre docentes.
- Cruces entre aulas.
- Cruces entre secciones.
- Mala distribución horaria.
- Dificultad para validar disponibilidad docente.
- Falta de una oferta académica clara para el estudiante.
- Dificultad para que el estudiante construya un horario personal sin conflictos.
- Uso ineficiente de recursos académicos.

## 1.3 Solución propuesta

OptiAcademic propone una aplicación web full-stack que:

1. Permite administrar datos académicos.
2. Genera horarios institucionales mediante un motor CSP.
3. Valida conflictos y restricciones.
4. Publica una oferta académica oficial.
5. Permite al estudiante generar alternativas de horario personal.
6. Recomienda la mejor opción según criterios de calidad.
7. Permite guardar y exportar el horario final.

## 1.4 Alcance funcional

El sistema incluye los siguientes módulos:

- Autenticación.
- Gestión de usuarios.
- Gestión de docentes.
- Gestión de estudiantes.
- Gestión de cursos.
- Gestión de aulas.
- Gestión de secciones.
- Generación de horarios institucionales.
- Diagnóstico de dominios CSP.
- Evaluación de calidad de horarios.
- Publicación de horarios institucionales.
- Oferta académica para estudiantes.
- Generación de horarios personales.
- Comparación de soluciones.
- Exportación de horario final a PDF.

## 1.5 Alcance académico

El sistema está orientado inicialmente a la carrera de **Ingeniería de Sistemas e Informática**, organizada en 10 ciclos académicos.

## 1.6 Fuera de alcance inicial

En la versión actual no se considera como alcance principal:

- Matrícula oficial con pagos.
- Integración con sistemas externos de la universidad.
- Firma digital de horarios.
- Aplicación móvil nativa.
- Notificaciones por correo institucional.
- Control de asistencia.
- Gestión de notas.
- Gestión financiera.
