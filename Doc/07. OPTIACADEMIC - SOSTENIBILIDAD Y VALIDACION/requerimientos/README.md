## Objetivo de esta documentación

Definir de forma ordenada los requerimientos que debe cumplir OptiAcademic para gestionar, generar, validar y publicar horarios académicos institucionales, así como permitir que los estudiantes generen horarios personales desde una oferta académica publicada.

## Alcance de requerimientos

Los requerimientos cubren:

- Gestión de usuarios.
- Gestión de docentes.
- Gestión de estudiantes.
- Gestión de cursos.
- Gestión de aulas.
- Gestión de secciones.
- Generación de horario institucional.
- Diagnóstico CSP.
- Publicación de oferta académica.
- Generación de horario estudiantil.
- Comparación de alternativas.
- Guardado de horario final.
- Exportación a PDF.
- Seguridad y control por roles.
