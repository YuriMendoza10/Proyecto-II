# 06. Cierre

Esta carpeta reúne documentación de cierre, conclusiones y entregables finales del proyecto.

## Contenido

- Conclusiones.
- Entregables finales.
- Evidencias finales.
- Lecciones aprendidas.
- Resumen de entrega.
- Archivos originales reorganizados en `entregables-originales/`.

La subcarpeta de entregables originales preserva documentos de la estructura anterior sin perder trazabilidad.

Los entregables organizados segun los criterios de evaluacion se encuentran en:

- [entregables-rubrica](entregables-rubrica/)
