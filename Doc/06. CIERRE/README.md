# 06. Cierre

Esta carpeta consolida la fase de control y cierre del proyecto OptiAcademic, correspondiente al curso Taller de Proyectos 2. La documentación se orienta a dejar evidencia formal, versionable y auditable del cierre del proyecto desde una perspectiva PMBOK, técnica y académica.

## Objetivo de la fase

Formalizar el cierre del proyecto mediante documentos que permitan verificar:

- Cumplimiento del alcance comprometido.
- Control de calidad y evidencias técnicas.
- Registro de riesgos, incidentes, defectos, impedimentos y supuestos.
- Lecciones aprendidas.
- Capacitación para operación, demostración y sustentación.
- Trazabilidad entre la consigna, los entregables y las evidencias.
