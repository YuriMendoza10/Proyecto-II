# 06. Contribucion A La Sostenibilidad

## Criterio Cubierto

Contribucion a la sostenibilidad del software.

## Que Revisar

La contribucion se evidencia en:

- menos datos transferidos;
- menos solicitudes repetidas;
- menor carga inicial;
- mejor rendimiento percibido;
- medicion de CO2 estimado;
- trazabilidad del consumo digital por funcionalidad.
