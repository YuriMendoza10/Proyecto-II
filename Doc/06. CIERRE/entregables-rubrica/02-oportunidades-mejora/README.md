# 02. Oportunidades De Mejora

## Criterio Cubierto

Identificacion de oportunidades de mejora.

## Que Revisar

La evidencia muestra oportunidades detectadas en:

- paginacion de listados grandes;
- cache de catalogos estables;
- reduccion de solicitudes HTTP repetidas;
- lazy loading en rutas React;
- optimizacion de consultas y reportes;
- lenguaje amigable en reportes para usuarios no tecnicos.
