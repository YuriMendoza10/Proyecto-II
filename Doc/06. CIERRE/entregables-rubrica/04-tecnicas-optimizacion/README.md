# 04. Tecnicas De Optimizacion

## Criterio Cubierto

Aplicacion de tecnicas especificas de optimizacion.

## Que Revisar

Tecnicas aplicadas o documentadas:

- paginacion;
- cache;
- lazy loading;
- reduccion de solicitudes HTTP;
- optimizacion de reportes;
- medicion de tiempos, bytes transferidos y CO2 estimado;
- optimizacion CSP para generacion de horarios.
