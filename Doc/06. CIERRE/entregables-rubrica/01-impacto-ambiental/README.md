# 01. Impacto Ambiental

## Criterio Cubierto

Analisis del impacto ambiental del software.

## Que Revisar

Esta seccion sustenta la identificacion de cinco o mas impactos ambientales relacionados con desarrollo, despliegue y uso de aplicaciones web:

- consumo de energia durante solicitudes HTTP;
- transferencia de datos innecesaria;
- procesamiento de reportes pesados;
- contenedores activos durante pruebas;
- almacenamiento y consultas sobre datos no optimizados;
- cargas iniciales grandes en frontend.
