# Entregables Organizados Segun La Rubrica

Esta carpeta organiza las evidencias finales del proyecto OptiAcademic segun los criterios de evaluacion de la rubrica "Desarrollo web responsable y reduccion del impacto ambiental en proyectos MERN".

OptiAcademic adapta la rubrica al stack real del proyecto: React, Vite, Tailwind CSS y Axios en frontend; FastAPI, SQLAlchemy y Alembic en backend; MySQL como base de datos; y Docker Compose como entorno local.

No se duplican archivos pesados. Cada subcarpeta enlaza a documentos y capturas ya existentes dentro de `Doc/`.
