# Evidencias de ejecucion

Las evidencias visuales de Semana 13 deben guardarse en:

- [evidencias](evidencias/)

## Evidencias recomendadas

- Captura de `py -m pytest -v` exitoso.
- Captura de `npm --prefix frontend run test` exitoso.
- Captura de `npm --prefix frontend run e2e` con pruebas exitosas o skips justificados.
- Captura de `npm --prefix frontend run build` exitoso.
- Captura de Swagger o `/docs` disponible.
- Captura de `/login` cargando sin pantalla blanca.
- Captura del reporte de sostenibilidad.
- Captura de consola sin errores criticos.

No se inventan capturas en esta carpeta. Solo se documenta donde deben almacenarse.
