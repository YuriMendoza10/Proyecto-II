Esta carpeta debe almacenar las capturas y archivos de evidencia generados durante la ejecucion de pruebas.

## Evidencias sugeridas

- Captura de pruebas backend con `py -m pytest -v`.
- Captura de pruebas frontend con `npm --prefix frontend run test`.
- Captura de pruebas e2e con `npm --prefix frontend run e2e`.
- Captura de build frontend exitoso.
- Captura de Docker Compose config exitoso.
- Capturas de rutas principales funcionando.

No se agregan imagenes ficticias. Las evidencias visuales deben corresponder a ejecuciones reales.
