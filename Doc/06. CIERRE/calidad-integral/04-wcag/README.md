# WCAG - Accesibilidad

Esta seccion prepara la validacion WCAG de OptiAcademic.

## Rutas principales a evaluar

- `/login`.
- `/admin/institutional-csp`.
- `/admin/schedule-view`.
- `/admin/reports/sustainability`.

## Criterios

- Contraste de colores.
- Navegacion con teclado.
- Foco visible.
- Labels en formularios.
- Textos alternativos.
- Botones accesibles.
- Mensajes de error.
- Estructura semantica.
- Modo claro/oscuro.
