# Evidencias WCAG

Guardar capturas reales de:

- Lighthouse Accessibility por ruta evaluada.
- Validacion manual con teclado.
- Foco visible en formularios y menus.
- Contraste en modo claro.
- Contraste en modo oscuro.
- Tablas y reportes legibles.
- Mensajes de error comprensibles.

No se deben inventar puntajes. Cada captura debe indicar ruta, fecha aproximada y herramienta usada.
