# Evidencias OWASP

Guardar aqui o enlazar desde aqui capturas reales de:

- `py -m pytest tests/test_security_core.py -v`.
- Login invalido con error controlado.
- Ruta protegida sin token devolviendo 401 o 403.
- `npm audit`.
- `pip-audit`.
- Revision de `.env` no versionado.
- Matriz de vulnerabilidades revisada.
- Riesgo residual actualizado.

No se deben inventar capturas. Toda imagen debe provenir de una ejecucion real del sistema o de comandos reales.
