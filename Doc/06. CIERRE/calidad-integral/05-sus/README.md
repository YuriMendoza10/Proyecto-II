# SUS - System Usability Scale

SUS permite medir la percepcion de usabilidad mediante un cuestionario de 10 preguntas con escala de 1 a 5.

## Aplicacion en OptiAcademic

Participantes sugeridos:

- Administrador.
- Coordinador.
- Docente.
- Estudiante.

## Flujo recomendado para evaluar

1. Iniciar sesion.
2. Revisar dashboard.
3. Consultar reportes.
4. Interactuar con generacion CSP o vista de horarios.
5. Revisar notificaciones o informacion academica.
6. Responder el instrumento SUS.

No se registran resultados inventados. La base debe llenarse con participantes reales.

## Estado actual

El instrumento SUS queda preparado para aplicacion real. Los resultados solo deberan completarse luego de aplicar el cuestionario a participantes reales o usuarios de prueba.

Mientras no existan respuestas reales, la seccion SUS debe marcarse como pendiente de aplicacion en la matriz de cumplimiento.
