# Calidad integral - SonarQube, OWASP, WCAG, SUS y Testing

Esta carpeta organiza la documentacion de calidad integral de OptiAcademic para la consigna de revision de aplicaciones Web modernas Full Stack.

OptiAcademic no usa MERN. El proyecto real usa React, Vite, Tailwind CSS, Axios, FastAPI, SQLAlchemy, Alembic, MySQL y Docker Compose. Por ello, la rubrica se aplica por equivalencia tecnica, manteniendo los mismos objetivos de seguridad, accesibilidad, usabilidad, mantenibilidad, verificabilidad y calidad arquitectonica.
