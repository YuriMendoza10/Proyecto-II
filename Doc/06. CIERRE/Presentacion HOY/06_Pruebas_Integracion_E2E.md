# Pruebas de Integración, Aceptación y E2E

## Pruebas de Integración

### Objetivo
Validar la interacción entre frontend, backend y base de datos.

### Herramientas
- FastAPI TestClient
- Pytest
- MSW
- React Testing Library

### Escenarios
- CRUD de entidades.
- Autenticación.
- Autorización.
- Respuestas HTTP.
- Persistencia de datos.

---

## Pruebas de Aceptación

### Herramienta
- Cypress

### Escenarios
- Inicio de sesión.
- Navegación.
- Gestión de información.
- Validaciones funcionales.

---

## Pruebas End-to-End

### Herramienta
- Playwright

### Escenarios
- Golden Path.
- Happy Path.
- Unhappy Path.
- Persistencia.
- Seguridad.
- Manejo de errores.

Garantizar que el sistema funcione correctamente desde la perspectiva del usuario final.
