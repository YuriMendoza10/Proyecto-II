# Pruebas Unitarias

## Objetivo
Validar funciones y lógica de negocio de manera aislada.

## Herramientas Utilizadas

### Frontend
- Vitest
- React Testing Library

### Backend
- Pytest
- FastAPI TestClient

## Casos Probados

### Reglas de Negocio
- Validación de disponibilidad docente.
- Validación de aulas.
- Restricciones académicas.

### Manejo de Errores
- Datos inválidos.
- Campos obligatorios.
- Excepciones controladas.

### Mocking
- MSW para simulación de APIs.
- Mocks y spies para dependencias.

## Beneficios
- Detección temprana de errores.
- Mayor mantenibilidad.
- Validación continua del sistema.
