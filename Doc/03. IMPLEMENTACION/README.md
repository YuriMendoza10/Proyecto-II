## Mejora de generacion institucional CSP

Se mejoro la vista de generacion institucional CSP incorporando una lista de horarios generados, evitando que el usuario dependa unicamente del ID manual para cargar horarios. La pantalla permite buscar, filtrar, ver detalle, cargar horarios existentes, revisar bloques y publicar cuando corresponde.
