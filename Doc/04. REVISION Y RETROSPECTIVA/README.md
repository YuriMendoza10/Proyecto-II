# 04. Revision Y Retrospectiva

Esta carpeta organiza la documentación de revisión de avances, observaciones y retrospectiva del proyecto.

## Contenido

- Revisión de avances.
- Observaciones del proceso.
- Retrospectiva del equipo.
- Mejoras detectadas.
- Validación parcial.
- Entregables parciales originales conservados en `entregables-parcial-originales/`.

La subcarpeta de entregables parciales conserva imágenes, capturas y documentos usados como evidencia del avance del proyecto.
