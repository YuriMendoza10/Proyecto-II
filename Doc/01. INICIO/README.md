# 01. Inicio

Esta carpeta reúne los documentos iniciales del proyecto. Presenta el contexto, la problemática, los objetivos, la visión del producto y la conformación del equipo.

## Contenido

- Presentación y enfoque del proyecto.
- Problema abordado y contexto inicial.
- Objetivos y visión del producto.
- Project Charter.
- Supuestos y restricciones.
- Integrantes del equipo.
- Requerimientos funcionales y no funcionales preliminares.
