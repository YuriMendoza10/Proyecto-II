## Reglas de Exposición

- Tiempo sugerido por integrante: **1 a 2 minutos**.
- No repetir información ya presentada.
- Mostrar evidencias concretas y mantener abiertas solo las rutas necesarias.
- No abrir `.docker.env`, `.env`, tokens, contraseñas ni credenciales.
- El video explicativo se realizará aparte y no es el tema principal de esta guía.
