## 5.3 Roles según Scrum

- **Product Owner:** Define prioridades del producto y valida entregables  
- **Scrum Master:** Facilita el proceso ágil y elimina impedimentos  
- **Equipo de Desarrollo:** Implementa funcionalidades y asegura calidad

## 5.4 Responsabilidades del Equipo

- Implementación del modelo CSP  
- Validación de datos y pruebas

## 5.5 Normas de Trabajo

- Uso de metodología Scrum con Sprints  
- Reuniones periódicas de seguimiento (Daily Scrum)  
- Revisiones de Sprint para validar avances  
- Retrospectivas para mejora continua  
- Trabajo colaborativo mediante control de versiones

## 5.6 Comunicación y Seguimiento

- Coordinación continua entre miembros del equipo  
- Uso de herramientas colaborativas (GitHub, etc.)
