# 🚀 Guía de Inicio - Proyecto OptiAcademic

Este archivo contiene los comandos necesarios para levantar el proyecto desde cero en una computadora local.

## 📌 Requisitos Previos
* MySQL Server (v8.0+) corriendo en el puerto 3306.
* Python 3.10+ instalado.
* Node.js y npm instalados.

---

## 🗄️ 1. Base de Datos (MySQL)
1. Asegurarse de que el servicio de MySQL esté iniciado.
2. La base de datos debe llamarse: `optiacademic_db`.
3. Credenciales configuradas en `backend/database.py`:
   - **User:** `root`
   - **Password:** `11111111`

---

## ⚙️ 2. Backend (Python + FastAPI)
cd frontend
# Instalar paquetes (solo la primera vez o si hay cambios)
npm install
# Iniciar servidor de desarrollo
npm run dev


🎨 3. Frontend (React + Vite)

cd frontend
