# Changelog

Los cambios relevantes de OptiAcademic se documentan en este archivo.

## [0.1.0] - 2026-05-26

### Agregado

- Aplicacion full-stack React + FastAPI + MySQL para gestion academica.
- Generador y validadores de horarios basados en CSP.
- Autenticacion JWT y vistas diferenciadas por roles.
- Entorno local con Docker Compose para frontend, backend y MySQL.
- Dashboard de impacto ambiental con persistencia de metricas HTTP.
- Escenario GreenFrame con Playwright para el flujo administrativo ambiental.
- Endpoint publico `GET /api/v1/sustainability` para consultar el ultimo reporte.
- Documentacion organizada para arquitectura, instalacion y presentacion.
