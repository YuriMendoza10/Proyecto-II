## 📌 Descripción

El sistema es una aplicación web orientada a la automatización de la planificación de horarios universitarios, dirigida a coordinadores académicos y personal administrativo. Permite gestionar cursos, docentes, aulas y disponibilidades de manera centralizada.

El problema que resuelve es la ineficiencia de los procesos manuales, los cuales generan conflictos de horarios, consumo excesivo de tiempo y errores en la asignación de recursos.

El alcance del PMV incluye el registro de información académica, la gestión de disponibilidad y la generación automática de horarios sin conflictos básicos, con el objetivo de validar la solución en un entorno real.
